# Модель статусів

## Канонічна послідовність (вказівка користувача 03.07.2026)

1. Стафіровка
2. Завантажений на авто / потяг  (вивіз від відправника → порт; export-бік)
3. На кордоні
4. Зданий в порт
5. Завантажений на судно
6. **В морі** (від завантаження на судно до вивантаження в порту прибуття)
7. Вивантажений в порту прибуття
8. Завантажений на авто / потяг  (від порту прибуття вглиб)
9. Прибув в сухій порт
10. Завантажений на авто  (остання миля від сухого порту)
11. Доставка отримувачу  (привезли авто, порожній ще не повернуто)
12. **Вантаж доставлено**  (порожній повернуто; фінальний, ЗАМОРОЖЕНО)

- «У дорозі» ПРИБРАНО — замінено на «Завантажений на потяг»/«Завантажений на авто» за видом транспорту.
- «Вантаж доставлено» замінив стару назву «Доставлено клієнту».

## Maersk-мапінг (rozbir.code.js) — за кодом+модою ОСТАННЬОЇ події

| Код події | Транспорт | Статус |
|---|---|---|
| LOAD / DEPA | VESSEL | В морі |
| ARRI | VESSEL | В морі (перевалка на воді) |
| DISC | VESSEL | Вивантажений в порту прибуття |
| GTOT / LOAD / DEPA | RAIL | Завантажений на потяг |
| GTOT / LOAD / DEPA | TRUCK | Завантажений на авто |
| ARRI / DISC | RAIL | Прибув в сухій порт |
| ARRI | TRUCK | Доставка отримувачу |
| **GTIN** | **TRUCK** | **Вантаж доставлено** (порожній повернуто) |
| GTIN | RAIL | Прибув в сухій порт |

**Заморозка:** якщо поточний статус = «Вантаж доставлено» — Розбір не перезаписує.

**Чому GTIN/TRUCK = доставлено:** у подіях Maersk поле `emptyIndicator` НЕ заповнюється, тож повернення порожнього визначаємо за останньою подією GTIN на авто (цикл GTOT→GTIN→GTOT→GTIN у кінці = видача + повернення порожнього).

## Стафіровка / На кордоні / Зданий в порт
Maersk для імпорту зазвичай НЕ дає цих подій — вносяться вручну або з листів. Для експортних відправлень частина буде.

## ETA / судно (той самий вузол)
- ETA порт: з останньої події ARRI. `eventClassifierCode=ACT` → «ETA порт (факт)»; `EST` → «ETA порт (план)» + лог зміни в «Зміни ETA (історія)».
- Судно/Вояж: з останньої VESSEL-події, лише якщо днів до ETA ≤ 7.
- Контейнер (лінія): усі equipmentReference; розбіжність із «Контейнер» → прапорець у «Звірка».
