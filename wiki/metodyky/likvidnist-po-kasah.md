# Методика: ліквідність по касах

## ⚠ Головне правило
**Каси НЕ взаємозамінні.** Не можна перекинути гроші з готівкової каси на банківський
рахунок довільно. Тому агрегатне «вільні гроші» по всіх касах разом — **без сенсу**.
Ліквідність аналізувати ПО ВИДАХ ОПЛАТИ (касах) окремо.

## Як рахувати
По кожній касі окремо таблицею:
```
Залишок  |  До сплати  |  Очікувані надходження  |  Баланс
```
Дефіцитні каси (Баланс < 0) підсвічувати червоним.

- У рахунках 1С є прив'язка до каси: доходні кол. 3 «Вид оплати», витратні кол. 8
  «Вид оплати» (100% заповнена) — див. [[mapping-1c]].
- Список кас — [[kasy]].

## Приклад висновку (ілюстрація, знімок 03.07 — НЕ актуальна цифра)
Кредиторка висіла на гривневому «Банк Юнітекс Ейч-Ді» (малий залишок), а гроші лежали
в касах USD → дефіцит по гривневому банку, попри профіцит по USD-касах.
Актуальні цифри — завжди в [[finzvit]].
