# Методика: заморожений капітал в угодах

Скільки наших грошей «зв'язано» в незакритих угодах.

## Формула (правильна, v8)
```
заморожено в угоді = max(0, ОПЛАЧЕНІ витрати по угоді − виставлені клієнту доходи)
```
- `оплачені витрати (paid_uo)` = `Сума УО − Борг` по витратних рахунках угоди.
- Вік заморозки — **від дати оплати постачальнику** (не від дати виставлення),
  зважено за оплаченою частиною.

## ⚠ Урок (чому саме оплачені витрати)
Рахувати за ВИСТАВЛЕНИМ рахунком постачальника — помилка: неоплачений рахунок
постачальника ≠ заморожені гроші (це кредиторка!). Інакше виходить подвійний рахунок.
Приклад: Кампус Коттон Клаб мав «заморожено» 7 500 через неоплачені Betterway №826/827 —
насправді це кредиторка, заморожено = 0.

## Заморожений капітал разом (для дашборда)
```
заморожено разом = дебіторка + аванси постачальникам + незакриті (оплачені) витрати по угодах
```

## Примітки
- Маржа по угодах клієнта — навмисно рахується за ВСІМА рахунками (і неоплаченими):
  це собівартість, не грошовий потік. Не плутати з замороженим капіталом.
- Конкретні суми на дату — у [[finzvit]], не тут.
