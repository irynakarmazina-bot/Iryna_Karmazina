# raw/ — сирі джерела

Тут лежать необроблені джерела, з яких Claude витягує факти у теми вікі.
**Не редагувати руками** — це «фото» інформації на момент потрапляння сюди.

## Поточні джерела засіву (Фаза A)
Вікі засіяна з журналу сесій, який фізично живе в корені репо:
- `../../MEMORY.md` — постійний блок + журнал сесій.
- `../../MEMORY_ARCHIVE.md` — архів старих секцій журналу.

Щоб не дублювати 448 рядків, на Фазі A вони не копіюються сюди, а лишаються
першоджерелом у корені. Наступні фази складатимуть сюди нові сирі матеріали:
- `exports_1c/` — вивантаження Експедитора (доходні/витратні/каси/угоди);
- `sessions/` — підсумки сесій Claude;
- `docs/` — ТЗ, листи, скріни.
