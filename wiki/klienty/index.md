# Клієнти

Дебіторка, терміни оплати, маржа. **Актуальні суми — у [[finzvit]]**, тут — стабільні
факти й останні відомі оцінки з поміткою дати. Методики: [[dso-dpo]], [[zamorozheno-v-ugodah]].

Аркуш «КЛІЄНТИ» звіту (методика узгоджена): дебіторка + вік (зважений), незакриті витрати
по відкритих угодах + вік, РАЗОМ зависає, дохід з 01.03, маржа по угодах, % зависає/дохід.
