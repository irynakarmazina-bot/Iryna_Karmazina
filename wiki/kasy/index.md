# Каси / види оплати

13 кас/рахунків у 3 валютах (UAH / USD / EUR). Джерело: довідник «Вид оплати» Експедитора.
**Каси НЕ взаємозамінні** — див. [[likvidnist-po-kasah]].

## Список (за скріном довідника)
- Банк Юнітекс Ейч-Ді — UAH / USD / EUR / Восток
- Банк Юнітекс Форвардинг
- Банк Cr String Cycle LLC (USD)
- Депозити USD
- Каси UAH / USD — UFWD і UHD
- Каса USD Украмарин
- Маерск USD

## Примітки
- У звіті [[finzvit]] залишки показуються по кожній валюті окремо (РАЗОМ UAH / USD / EUR);
  ліквідність і календар — в У.О.
- Прив'язка рахунку до каси: доходні кол. 3, витратні кол. 8 (див. [[mapping-1c]]).
