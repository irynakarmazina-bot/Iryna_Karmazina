# Індекс вікі

Каталог усіх сторінок (1 рядок на кожну). Оновлювати при додаванні/зміні сторінки.

## Службові
- [[overview]] — карта: проєкти, гілки, ключові факти, відкриті питання.
- [[baza-znan]] — Notion «UNITEX \| База знань»: бізнес-довідка + нотатки (шар Notion гібриду).
- [[README]] — правила ведення вікі (схема).
- `log.md` — хронологія змін вікі.
- `raw/` — сирі джерела (першоджерело засіву: MEMORY.md).

## Проєкти (`proyekty/`)
- [[bot]] — Telegram-бот на Claude; VPS, autodeploy, Git Relay.
- [[finzvit]] — Оперативний фінансовий звіт з 1С (Google Sheets + дашборд).
- [[dyspetcheryzatsiya]] — міні-CRM вантажів (n8n, Maersk-трекінг, статуси).
- [[1c-api]] — доступ до 1С Експедитор, ТЗ на API, критичні факти про дані.

## Методики (`metodyky/`)
- [[zamorozheno-v-ugodah]] — заморожений капітал в угодах (за оплаченими витратами).
- [[dso-dpo]] — реальні терміни оплати (замість «планової дати» 1С).
- [[likvidnist-po-kasah]] — ліквідність по касах (каси не взаємозамінні).
- [[kasovyy-rozryv]] — прогноз касового розриву на 30 днів.
- [[mapping-1c]] — мапінг колонок експортів Експедитора.
